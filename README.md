# Multi-Agent System 🚀

An advanced multi-agent orchestrator (Planner, Executor, Critic) built with Node.js and React, powered by the Groq SDK.

## ✨ Features
- **Multi-Agent Workflow**: Autonomous collaboration between a Planner, Executor, and Critic.
- **Web Search Tool**: Keyless web searching via DuckDuckGo.
- **File System Integration**: Agents can read and write local files.
- **Conversation Memory**: Persistent chat history with multi-turn capabilities.
- **Modern UI**: Sleek, glassmorphic React frontend with real-time event streaming.

## 🛠 Tech Stack
- **Backend**: Node.js, Express, Groq SDK, Axios, Cheerio.
- **Frontend**: React, Vite, React Markdown.
- **Model**: Llama-3.1-8b-instant (optimized for speed/free-tier).

## 🚀 Getting Started

### 1. Prerequisites
- [Node.js](https://nodejs.org/) (v16+)
- A [Groq API Key](https://console.groq.com/keys)

### 2. Setup
Clone the repository and install dependencies:

```bash
# Install Backend dependencies
cd backend
npm install

# Install Frontend dependencies
cd ../frontend
npm install
```

### 3. Configuration
Create a `.env` file in the `backend` folder:
```env
GROQ_API_KEY=your_groq_api_key_here
PORT=5000
```

### 4. Running the App
Start both servers simultaneously:

**Backend:**
```bash
cd backend
npm run dev
```

**Frontend:**
```bash
cd frontend
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) to start chatting!

## 📂 Project Structure
- `backend/`: Express server, agent logic, and tool definitions.
- `frontend/`: React application and chat UI.
- `outputs/`: (Optional) Directory for files generated by the agents.
